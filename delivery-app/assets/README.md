# SwiftDrop Assets

Place any custom images, icons, or other static assets here.

## Current Setup
- All icons are served via Iconify CDN (no downloads needed)
- Leaflet.js maps via CDN
- Google Fonts via CDN

## Font Used
- Syne (Display/Headings)
- DM Mono (Monospace/Labels)
- Manrope (Body text)
